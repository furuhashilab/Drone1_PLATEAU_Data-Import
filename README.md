# Drone1_PLATEAU_Data-Import
## ハッカソン内容
画像系生成AIとして頭角を現してきている、Google Gemini の **Nano Banana** を活用してOpenStreetMap に関する特定テーマのグラレコ風マニュアルを作成する。
Drone1として、 ***PLATEAUのデータインプットマニュアル*** に取り組んだ。
